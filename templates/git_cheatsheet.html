Basic Commands

  • git init - Initialize a new Git repository
  • git clone [url] - Clone a repository
  • git add [file] - Add file to staging area
  • git commit -m "[message]" - Commit changes
  • git status - Check repository status

Branching & Merging

  • git branch - List branches
  • git branch [branch-name] - Create a new branch
  • git checkout [branch-name] - Switch to a branch
  • git merge [branch] - Merge a branch into current branch
  • git rebase [branch] - Rebase current branch onto another branch
üåê

Remote Repositories

  • git remote add [name] [url] - Add a remote repository
  • git fetch [remote] - Fetch changes from remote
  • git pull [remote] [branch] - Pull changes from remote
  • git push [remote] [branch] - Push changes to remote
  • git remote -v - List remote repositories

Undoing Changes

  • git reset [file] - Unstage a file
  • git reset --hard - Reset to last commit
  • git revert [commit] - Revert a commit
  • git checkout -- [file] - Discard changes in working directory
  • git clean -fd - Remove untracked files

Log & History

  • git log - View commit history
  • git log --oneline - Compact commit history
  • git log --graph - View commit history with graph
  • git diff [commit1] [commit2] - Diff between two commits
  • git show [commit] - Show changes in a commit

Stashing

  • git stash - Stash changes
  • git stash list - List stashes
  • git stash apply - Apply last stash
  • git stash drop - Delete last stash
  • git stash pop - Apply and delete last stash
